# Applied LLM Engineering

Welcome to my repository for Applied LLM Engineering projects! This root directory contains a collection of projects leveraging Large Language Models, LangChain, and advanced RAG architectures.

## Projects

1. **Hybrid Retrieval System**: A robust retrieval pipeline exploring keyword and semantic search combining PyMuPDF, BM25, and vector stores for highly accurate document retrieval.
2. **Memory System Deep Dive**: Development of stateful AI interactions using LangChain and Groq, exploring conversation buffers, summaries, and memory management.
3. **Production-Ready RAG**: A robust Retrieval-Augmented Generation system incorporating FAISS, vector databases, and evaluation datasets (ready for deployment).
4. **Tool-Based Agent**: A ReAct-based agent system allowing LLMs to interact with external tools and APIs for intelligent decision-making and calculations.

Each project contains its own `README.md` with instructions on how to install dependencies and run the application.

## Global Setup

Most projects require an environment with Python and an API key (e.g., Groq, OpenAI). Be sure to configure the `.env` file within the respective project directory before running it.
